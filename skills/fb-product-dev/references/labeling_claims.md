# FDA Labeling & Claims for Beverages

## Overview

The label is both a marketing asset and a legal document. Every word, number, and graphic on a beverage package is governed by a layered regulatory framework: federal (FDA, FTC, USDA for organic, TTB for alcohol), state (California Prop 65, NY sodium, state-specific labeling), and voluntary programs (Non-GMO Project, Fair Trade, Rainforest Alliance). Getting labeling wrong is one of the most expensive mistakes in beverage NPD -- a full reprint of labels or packaging can cost $50K-$200K and delay launch by 8-16 weeks.

## Nutrition Facts Panel (21 CFR 101.9)

### Serving Size (RACC)

| Beverage Type | RACC | Notes |
|---|---|---|
| Beverages (most) | 240 mL (8 fl oz) | Includes juices, teas, waters, CSDs, sports drinks |
| Beverages sold as single serve | Entire container if <= 200% RACC (480 mL / 16 fl oz) | One serving = entire container |
| Beverages 200-300% RACC (480-720 mL) | Dual column labeling required | Column 1: per serving (RACC); Column 2: per container |
| Beverages > 300% RACC (> 720 mL) | Per serving (RACC) only | Multi-serve container |
| Concentrated beverages (to be diluted) | Amount to make 240 mL as consumed | "As prepared" basis |

### Dual Column Labeling

Required for packages containing 200-300% RACC. Example: a 16 fl oz (473 mL) can of juice.

| Nutrient | Per Serving (8 fl oz) | Per Container (16 fl oz) |
|---|---|---|
| Calories | 120 | 240 |
| Total Sugars | 28g | 56g |
| Added Sugars | 0g (0% DV) | 0g (0% DV) |

The dual column format was mandated under the 2016 Nutrition Facts update to address the reality that many single-serve packages are consumed in one sitting.

### Rounding Rules (Selected)

| Nutrient | Amount | Round To |
|---|---|---|
| Calories | < 5 | 0 |
| Calories | 5-50 | Nearest 5 |
| Calories | > 50 | Nearest 10 |
| Total fat | < 0.5g | 0g |
| Total fat | 0.5-5g | Nearest 0.5g |
| Total fat | > 5g | Nearest 1g |
| Total sugars | < 0.5g | 0g |
| Total sugars | 0.5-1g | "less than 1g" or nearest 1g |
| Total sugars | > 1g | Nearest 1g |
| Added sugars | < 0.5g | 0g |
| Sodium | < 5mg | 0mg |
| Sodium | 5-140mg | Nearest 5mg |
| Sodium | > 140mg | Nearest 10mg |
| Vitamins/minerals | All ranges | Nearest 2% DV (if 2-10%); nearest 5% DV (if 10-50%); nearest 10% DV (if >50%) |

### Key Nutrition Facts Requirements

- **Added Sugars:** Must be declared; includes sugars added during processing (sucrose, HFCS, honey, fruit juice concentrates used as sweeteners) but NOT sugars naturally present in fruit/vegetable juice
- **Vitamin D and Potassium:** Now mandatory on the panel (replaced Vitamins A and C, which are now voluntary)
- **Daily Values:** Updated in 2016; e.g., Sodium DV = 2,300mg, Added Sugars DV = 50g, Vitamin C DV = 90mg
- **Footnote:** Only the abbreviated footnote is required: "The % Daily Value tells you how much a nutrient in a serving of food contributes to a daily diet. 2,000 calories a day is used for general nutrition advice."

## Ingredient Statement (21 CFR 101.4)

### Rules

| Rule | Detail |
|---|---|
| Descending order of predominance | By weight at time of manufacture |
| Water declaration | Must be declared by its common name ("water" or "purified water"); listed in order of predominance |
| Sub-ingredients | Ingredients that are themselves multi-component (e.g., flavor preparations) must list sub-ingredients in parentheses |
| Common or usual names | Use 21 CFR 101.4 names; e.g., "sugar" not "cane sugar" (though "cane sugar" is also acceptable) |
| Colors | Certified colors by name and number (e.g., "Red 40"); exempt colors by common name (e.g., "beet juice (for color)", "turmeric") |
| Flavors | "Natural flavor," "artificial flavor," or "natural and artificial flavor" per 21 CFR 101.22 |
| Incidental additives | Processing aids present at insignificant levels may be omitted (but allergens must always be declared) |
| Chemical preservatives | Must declare functional purpose: e.g., "potassium sorbate (preservative)" or "sodium benzoate (to preserve freshness)" |

### Juice Percentage Declaration (21 CFR 101.30)

| Scenario | Label Requirement |
|---|---|
| 100% juice, single juice | "100% [fruit] juice" -- no % required on information panel |
| 100% juice, blend | "100% juice" on front; individual juices listed in ingredient statement |
| < 100% juice, contains juice | Must declare "Contains __% juice" on information panel |
| < 100% juice, juice named on front | If juice flavor is named (e.g., "Orange Flavored Beverage"), "Contains __% juice" must appear |
| 0% juice, fruit flavor only | Not required to declare % but must not imply juice content; "flavored" must be prominently displayed |
| Juice from concentrate | "from concentrate" or "reconstituted" must appear on label |
| Not from concentrate (NFC) | May state "not from concentrate" |

### Naming Conventions (Juice Beverages)

| Name | Juice Content | CFR Basis |
|---|---|---|
| "[Fruit] Juice" | 100% juice | 21 CFR 101.30 |
| "[Fruit] Juice from Concentrate" | 100% juice, reconstituted | 21 CFR 101.30 |
| "[Fruit] Juice Drink" | < 100% juice (typically 10-99%) | Common or usual name |
| "[Fruit] Juice Cocktail" | < 100% juice (typically 10-50%) | Common or usual name |
| "[Fruit] Flavored Drink" | May contain 0% juice; flavor derived from source other than named fruit | Must clarify on label |
| "Sparkling [Fruit] Beverage" | Variable; must declare % if juice present | Careful with consumer expectation |

## Allergen Labeling (FALCPA + FASTER Act)

### Major Allergens (as of 2023)

1. Milk
2. Eggs
3. Fish (specify species)
4. Crustacean shellfish (specify species)
5. Tree nuts (specify type: almond, cashew, walnut, etc.)
6. Peanuts
7. Wheat
8. Soybeans
9. **Sesame** (added by FASTER Act, effective January 1, 2023)

### Declaration Methods

- **"Contains" statement:** Separate statement immediately after or adjacent to ingredient list: "Contains: milk, soy"
- **Parenthetical in ingredient list:** e.g., "whey protein concentrate (milk), lecithin (soy)"
- **Either method is acceptable;** "Contains" statement is more common and considered best practice by most regulatory consultants

### Precautionary Allergen Labeling (PAL)

- "May contain [allergen]" or "Produced in a facility that also processes [allergen]" -- these are VOLUNTARY and NOT regulated by FDA
- Should be based on actual cross-contact risk assessment, not used as a blanket disclaimer
- Some retailers (Whole Foods) scrutinize excessive PAL statements

## Nutrient Content Claims (21 CFR 101.13, 101.54-101.69)

### Calorie Claims

| Claim | Requirement |
|---|---|
| Calorie free | < 5 kcal per RACC |
| Low calorie | <= 40 kcal per RACC |
| Reduced calorie | >= 25% fewer calories than reference food |
| Light/lite (calories) | 1/3 fewer calories than reference food |

### Sugar Claims

| Claim | Requirement |
|---|---|
| Sugar free | < 0.5g sugar per RACC; no ingredient that is a sugar or generally understood to contain sugars |
| No added sugars | No sugars or sugar-containing ingredients added during processing; must state if not "low calorie" or "reduced calorie" |
| Reduced sugar | >= 25% less sugar than reference food |

**Critical note on "No Added Sugars":** Fruit juice concentrate used as a sweetener IS an added sugar. Fruit juice concentrate used for flavor at levels consistent with normal juice content is not. FDA scrutinizes this distinction closely.

### Sodium Claims

| Claim | Requirement |
|---|---|
| Sodium free | < 5mg per RACC |
| Very low sodium | <= 35mg per RACC |
| Low sodium | <= 140mg per RACC |
| Reduced sodium | >= 25% less sodium than reference food |

### Vitamin/Mineral Claims

| Claim | Requirement |
|---|---|
| Good source | 10-19% DV per RACC |
| High / excellent source / rich in | >= 20% DV per RACC |
| More / enriched / fortified | >= 10% DV more than reference food per RACC |

**Example:** Vitamin C DV = 90mg. A beverage with 15mg Vitamin C per 8 fl oz serving (16.7% DV) can claim "Good source of Vitamin C." At 18mg (20% DV), it qualifies for "Excellent source."

## Health Claims & Structure/Function Claims

### Authorized Health Claims (21 CFR 101.72-101.83)

FDA has authorized a limited number of health claims linking a nutrient to a disease or health condition. Relevant to beverages:

| Claim Topic | Example Claim Language | Conditions |
|---|---|---|
| Calcium and osteoporosis | "Adequate calcium throughout life, as part of a well-balanced diet, may reduce the risk of osteoporosis" | >= 20% DV calcium per RACC; bioavailable form |
| Potassium and blood pressure | "Diets containing foods that are a good source of potassium and low in sodium may reduce the risk of high blood pressure and stroke" | >= 350mg potassium, <= 140mg sodium, <= 3g fat per RACC |

### Qualified Health Claims

- Supported by some but not conclusive scientific evidence
- Require specific qualifying language (e.g., "Some scientific evidence suggests...")
- Must be petitioned to FDA; no pre-market approval required but enforcement discretion letters guide permissible language
- Example: Green tea and cancer risk reduction (qualified)

### Structure/Function Claims (DSHEA, applied to foods)

- Describe the role of a nutrient in maintaining normal structure or function: "Calcium builds strong bones," "Antioxidants help maintain cell integrity"
- **Do NOT require FDA pre-approval for conventional foods** (unlike dietary supplements, which require 30-day notification)
- **Must NOT claim to diagnose, treat, cure, or prevent any disease**
- No disclaimer required for conventional food structure/function claims (unlike supplements)
- **Common beverage examples:** "Supports hydration," "Helps maintain energy levels," "Supports immune health"

**Red lines (these are drug claims, not structure/function):**
- "Reduces inflammation" (disease treatment)
- "Prevents colds" (disease prevention)
- "Lowers blood sugar" (disease treatment)

## "Natural" Labeling

FDA has not issued a formal regulation defining "natural" but has a longstanding policy:

- **Guidance:** "Nothing artificial or synthetic (including color additives regardless of source) has been included in, or has been added to, a food that would not normally be expected to be in that food"
- **Minimally processed:** Processing that does not fundamentally alter the raw product
- **What qualifies:** Pasteurization, fermentation, drying, roasting, freezing
- **What does NOT qualify:** Artificial flavors, synthetic colors (FD&C), synthetic preservatives, high-fructose corn syrup (debated)
- **Citric acid:** Commonly used in beverages; synthetic citric acid technically problematic for "natural" claims, but widely used. Many brands use "produced by fermentation" citric acid.
- **"All natural":** Stronger claim; every ingredient must be natural-origin. Higher scrutiny; litigation target.

**Litigation risk:** "Natural" is the #1 source of food labeling class-action lawsuits. Claims involving citric acid, "natural flavors" with synthetic solvents/carriers, or GMO-derived ingredients have all been litigated. Consult counsel before finalizing.

## Organic Labeling (USDA NOP, 7 CFR 205)

| Tier | Requirement | USDA Seal? | Label Placement |
|---|---|---|---|
| "100% Organic" | All ingredients certified organic (excluding water and salt) | Yes | Principal display panel |
| "Organic" | >= 95% organic ingredients (by weight, excluding water and salt) | Yes | Principal display panel |
| "Made with Organic [ingredient]" | >= 70% organic ingredients | No (may use certifier's seal) | Principal display panel |
| < 70% organic | May list organic ingredients in ingredient statement only | No | Ingredient statement only |

### Organic Certification Process

1. Apply to USDA-accredited certifying agent (QAI, CCOF, Oregon Tilth, etc.)
2. Submit Organic System Plan (OSP): ingredient sourcing, handling procedures, pest management, recordkeeping
3. Annual inspection of facility (and co-manufacturer, if applicable)
4. Organic certificates must be current for all organic ingredients (verify with suppliers)
5. Certification cost: $2,000-$10,000/year depending on complexity
6. **Co-man must also be certified** (or operate under your certification's scope)

## Non-GMO Project Verified

- Third-party certification by the Non-GMO Project
- Requires traceability and testing for all "high-risk" ingredients (corn, soy, sugar beet, canola, alfalfa, papaya, summer squash, and their derivatives)
- Testing threshold: < 0.9% GMO content per ingredient
- Process: application, documentation review, testing, annual re-verification
- Cost: $2,000-$15,000 annually depending on SKU count and ingredient complexity
- **Marketing value:** Consumers consistently rank Non-GMO as a top-5 purchase driver in natural channel

## Clean Label Considerations

"Clean label" is a marketing concept, not a regulatory category. Consumer expectations:

| Consumer Expectation | Implication for Formulation |
|---|---|
| Recognizable ingredients | Avoid technical names; e.g., "vitamin C" not "ascorbic acid" (though CFR requires the standard name in ingredient statement) |
| Short ingredient list | Target 5-10 ingredients for simple beverages |
| No artificial preservatives | Use clean-label preservation: pH < 4.0, high Brix, HPP, or natural antimicrobials (citric acid, rosemary extract) |
| No artificial sweeteners | Aspartame, sucralose, acesulfame-K out; stevia, monk fruit, allulose, erythritol in (though some consumers question these too) |
| No artificial colors | FD&C dyes out; fruit/vegetable juice for color, turmeric, beet, butterfly pea flower in |
| Transparent sourcing | Country of origin, direct trade, farmer stories |

## California Proposition 65

### Overview

Prop 65 (Safe Drinking Water and Toxic Enforcement Act of 1986) requires businesses to warn Californians about significant exposures to chemicals that cause cancer, birth defects, or reproductive harm.

### Beverages: Common Trigger Chemicals

| Chemical | Source in Beverages | Warning Trigger |
|---|---|---|
| Lead | Naturally present in some juice concentrates, botanical extracts | > 0.5 ug/day (MADL for reproductive toxicity) |
| Cadmium | Cocoa, some botanical extracts | > 4.1 ug/day |
| Acrylamide | Formed during roasting (coffee, certain grain-based beverages) | > 0.2 ug/day (NSRL) |
| 4-MeI (4-methylimidazole) | Caramel color (Class III and IV) | > 29 ug/day |
| Arsenic (inorganic) | Rice-based beverages, some juice concentrates | > 10 ug/day |
| BPA | Can linings (though most have transitioned to BPA-NI) | Listed reproductive toxicant |

### Warning Requirements

- **Short-form warning:** "WARNING: Cancer and Reproductive Harm - www.P65Warnings.ca.gov"
- **Long-form warning:** Names the specific chemical(s)
- Must appear on product label if sold in California OR at point of sale (website, shelf tag)
- **Safe harbor levels:** If exposure is below the NSRL (cancer) or MADL (reproductive), no warning required -- but brand must have testing data to support this
- **Best practice:** Test every ingredient lot for Prop 65 chemicals; establish specifications with suppliers; retain analytical records

## State-Specific Labeling Requirements

| State | Requirement | Applicable Products |
|---|---|---|
| California | Prop 65 warnings (see above) | All products sold in CA |
| California | CDPH registration for acidified/canned beverages | All shelf-stable beverages manufactured in or for sale in CA |
| New York City | Sodium warning icon on menu boards (>= 2,300mg) | Primarily foodservice; packaged beverages rarely triggered |
| Vermont | Required GMO labeling (superseded by federal NBFDS) | Covered by federal law as of 2022 |
| Federal (NBFDS) | National Bioengineered Food Disclosure Standard | Products containing bioengineered ingredients must disclose via text, symbol, QR code, or phone number |
| Multiple states | Bottle deposit laws (CA, CT, HI, IA, MA, ME, MI, NY, OR, VT) | Deposit amounts $0.05-$0.10; must be labeled on package |
| Multiple states | Recycling/recyclability labeling (CA SB 343, OR) | Chasing arrows symbol restrictions; "recyclable" claim must be substantiated |

## Beverage-Specific Labeling Checklist

Before sending a beverage label to print, verify:

- [ ] **Product identity / statement of identity** -- prominently displayed; accurate (e.g., "Sparkling Lemon Flavored Water" not "Sparkling Lemon Water" if no lemon juice)
- [ ] **Net quantity of contents** -- fluid ounces and metric (mL); lower 1/3 of PDP; correct type size per 21 CFR 101.105
- [ ] **Nutrition Facts panel** -- correct format (updated 2016 format), correct serving size, dual column if 200-300% RACC
- [ ] **Ingredient statement** -- descending order, correct names, sub-ingredients, allergens, colors, flavors, preservatives
- [ ] **Allergen declaration** -- "Contains" statement if any major allergen present
- [ ] **Juice percentage** -- declared if juice is present and/or implied by name or imagery
- [ ] **Nutrient content claims** -- meet criteria per 21 CFR 101.13 and specific nutrient sections; comparative claims name reference food
- [ ] **Structure/function claims** -- do not cross into disease claims; language reviewed by regulatory counsel
- [ ] **Organic labeling** -- correct tier, certifier name and address, USDA seal used only if >= 95%
- [ ] **Non-GMO / bioengineered disclosure** -- compliant with NBFDS if applicable
- [ ] **"Natural" claim** -- supportable for every ingredient and process; litigation risk assessed
- [ ] **Prop 65** -- testing supports no warning required, OR warning is on label
- [ ] **Bottle deposit** -- state-specific deposit amounts listed for all applicable states in distribution
- [ ] **Barcode / UPC** -- assigned, correctly printed, scannable (verify with barcode scanner)
- [ ] **Manufacturing codes** -- date coding format agreed with co-man (typically best-by date in MM/DD/YY or Julian format + plant code + line code)
- [ ] **Country of origin** -- "Product of [country]" if required (imported products)
- [ ] **Distributor / manufacturer name and address** -- required on all food labels; "Manufactured for [Brand], [City, State, ZIP]"
- [ ] **Kosher / Halal certification symbols** -- only if certified; must match certificate
- [ ] **Recycling instructions** -- How2Recycle label if participating; chasing arrows only if substantiated per state law
- [ ] **Legal review** -- outside regulatory counsel or experienced in-house team has reviewed final artwork

## Common Labeling Mistakes in Beverage NPD

| Rank | Mistake | Consequence |
|---|---|---|
| 1 | Incorrect serving size or RACC | FDA warning letter; retailer rejection; class action risk |
| 2 | Missing dual column for single-serve 200-300% RACC | Non-compliant Nutrition Facts panel |
| 3 | "No Added Sugars" with juice concentrate used as sweetener | FTC/FDA scrutiny; lawsuit (multiple class actions in recent years) |
| 4 | Structure/function claim that crosses into disease claim | FDA warning letter; required label change |
| 5 | Missing or incorrect allergen declaration | FDA recall (Class I if undeclared major allergen); serious liability |
| 6 | "Natural" claim with synthetic ingredient | Class action lawsuit; settlements averaging $1-5M |
| 7 | Prop 65 chemicals above safe harbor without warning | Prop 65 bounty hunter lawsuit; $2,500/day penalty |
| 8 | Juice percentage not declared or incorrect | FDA warning letter; consumer deception claim |
| 9 | Organic claim without certification or on wrong tier | USDA NOP enforcement; $11,000 penalty per violation |
| 10 | Incorrect rounding on Nutrition Facts | Technical violation; usually caught in retailer compliance review |

## Regulatory Review Timeline

| Activity | When in NPD | Duration |
|---|---|---|
| Initial regulatory scan | Stage 1 (Scoping) | 1-2 weeks |
| Claims strategy development | Stage 2 (Business Case) | 2-4 weeks |
| Draft label regulatory review | Stage 3 (Development) | 2-4 weeks |
| Nutritional analysis (lab) | Stage 3 (formula lock) | 2-3 weeks |
| Final artwork regulatory review | Stage 3-4 (pre-print) | 1-2 weeks |
| Legal review | Stage 3-4 (pre-print) | 1-2 weeks |
| Prop 65 testing | Stage 3-4 | 2-3 weeks |
| Process authority / FDA filing | Stage 3-4 | 2-6 weeks |
| State registration (e.g., CDPH) | Stage 4 (pre-commercial) | 4-12 weeks |

**Key principle:** Start regulatory work at Stage 2, not Stage 4. Claims strategy shapes formulation, and formulation determines what you can claim. These are interdependent -- not sequential.
