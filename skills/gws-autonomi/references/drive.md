## File Operations

### Upload File
```bash
# Upload with metadata
gws drive +upload ./report.pdf --name "Batch-Report-2847" --parents "FOLDER_ID"

# Upload with MIME type specification
gws drive files create \
  --params '{"uploadType": "multipart"}' \
  --json '{"name": "QC-Results-W12.xlsx", "parents": ["FOLDER_ID"]}' \
  --upload ./qc_results.xlsx
```

### Download File
```bash
# Download binary file
gws drive files get --params '{"fileId": "FILE_ID", "alt": "media"}' > output_file.pdf

# Export Google Doc as PDF
gws drive files export --params '{"fileId": "DOC_ID", "mimeType": "application/pdf"}' > report.pdf

# Export Google Sheet as CSV
gws drive files export --params '{"fileId": "SHEET_ID", "mimeType": "text/csv"}' > data.csv
```

### Copy File (template duplication)
```bash
gws drive files copy --params '{"fileId": "TEMPLATE_ID"}' \
  --json '{"name": "Weekly Report - W12", "parents": ["FOLDER_ID"]}'
```

### Delete / Trash
```bash
# Move to trash (recoverable)
gws drive files update --params '{"fileId": "FILE_ID"}' --json '{"trashed": true}'
```

## Folder Management

### Create Folder
```bash
gws drive files create --json '{
  "name": "March-2026-QC-Reports",
  "mimeType": "application/vnd.google-apps.folder",
  "parents": ["PARENT_FOLDER_ID"]
}'
```

### List Folder Contents
```bash
gws drive files list --params '{
  "q": "'\''FOLDER_ID'\'' in parents and trashed = false",
  "orderBy": "modifiedTime desc",
  "pageSize": 50
}'
```

## Tenant Workspace Setup

Run `scripts/setup_tenant_workspace.py` on new tenant onboarding. It creates:

```
Autonomi-{TenantName}/
├── Procurement/
│   ├── Purchase-Orders/          # PO PDFs generated by Procurement Agent
│   ├── Supplier-Contracts/       # Uploaded supplier agreements
│   ├── COAs/                     # Certificates of Analysis
│   └── Supplier-Scorecards/      # Monthly performance reports
├── Quality/
│   ├── Batch-Reports/            # Per-batch QC summary docs
│   ├── Hold-Notices/             # Quality hold documentation
│   ├── Lab-Results/              # Uploaded lab test results
│   └── Audit-Documents/          # SQF/BRC/FSMA audit prep
├── Production/
│   ├── Schedules/                # Weekly/monthly production plans
│   ├── Yield-Reports/            # Production efficiency analysis
│   └── BOM-Exports/              # Recipe/BOM documentation
├── Compliance/
│   ├── FSMA-204/                 # CTE/KDE records, trace reports
│   ├── Certifications/           # SQF, Organic, Kosher, etc.
│   ├── Recall-Records/           # Mock and real recall documentation
│   └── Training-Records/         # Employee food safety training
├── Finance/
│   ├── Invoices/                 # BILL.com invoice copies

│   └── AP-AR-Exports/            # Payment reconciliation exports

    ├── Decision-Logs/            # Weekly agent decision summaries
    ├── Anomaly-Reports/          # Flagged unusual patterns
    └── Performance-Dashboards/   # Agent performance metrics
```

## Search Patterns

```bash
# Find all COAs uploaded this month
gws drive files list --params '{
  "q": "name contains '\''COA'\'' and modifiedTime > '\''2026-03-01'\'' and trashed = false"
}'

# Find files by MIME type (PDFs only)
gws drive files list --params '{
  "q": "mimeType = '\''application/pdf'\'' and '\''FOLDER_ID'\'' in parents"
}'

# Full-text search across document content
gws drive files list --params '{
  "q": "fullText contains '\''allergen'\'' and trashed = false"
}'

# Find files shared with specific user
gws drive files list --params '{
  "q": "'\''user@example.com'\'' in readers or '\''user@example.com'\'' in writers"
}'
```
