# Formulation Lab Knowledge Domain

The Formulation Lab is Autonomi's R&D dashboard for beverage product development and commercialization. LynZ operates as the voice/chat assistant within the lab, controlling navigation, CRUD operations, and providing deep food science guidance.

## Reference Files

| File | Description |
|------|-------------|
| `references/dashboard_navigation.md` | All navigation commands, view routes, tab switching, platform toggles |
| `references/crud_operations.md` | Full CRUD command reference for clients, projects, recipes, trials, ingredients, suppliers, scale-up, versions, specs, gates, timeline, risks, integrations, feedback |
| `references/portfolio.md` | Current recipe portfolio (8 recipes), client roster (4 clients), project hierarchy |
| `references/view_knowledge.md` | What each dashboard tab/view does — integrations, workspace, ingredients, formula, trials, analytics, scale-up, versions, allergens, specs, gates, timeline, risks |

## How to Use This Skill

1. When LynZ needs to navigate the formulation lab → fetch `dashboard_navigation.md`
2. When LynZ needs to create/edit/delete anything → fetch `crud_operations.md`
3. When LynZ needs context about current recipes/clients → fetch `portfolio.md`
4. When LynZ needs to explain what a view shows → fetch `view_knowledge.md`
