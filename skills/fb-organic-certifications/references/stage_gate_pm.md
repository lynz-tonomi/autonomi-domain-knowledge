## 1. Stage-Gate Framework Overview

### Philosophy
Certification projects fail most often from project management failures, not technical failures. Missed supplier cert renewals, undisclosed scope changes, lapsed records, and untrained staff cause more suspensions than actual fraud or contamination events. The stage-gate model imposes discipline: you cannot advance to the next gate until you've met the entry criteria.

### The 7 Gates (G0–G6)

| Gate | Name | Purpose | Typical Duration |
|------|------|---------|-----------------|
| G0 | Scoping & Feasibility | Define the certification goal, business case, and rough plan | 2-4 weeks |
| G1 | Gap Assessment Complete | Understand the current state vs. certification requirements | 3-6 weeks |
| G2 | Program Build Complete | Build all documentation, SOPs, training, supplier programs | 8-16 weeks |
| G3 | Internal Audit Passed | Verify readiness through internal audit | 2-4 weeks |
| G4 | Application Submitted | Select certifier, submit application, pay fees | 2-4 weeks |
| G5 | Inspection/Audit Passed | Complete external audit, submit corrective actions | 4-8 weeks |
| G6 | Certificate Issued | Receive certificate, complete label review, communicate | 2-4 weeks |

**Total typical range: 6-12 months** for most certifications (organic handler can be faster if documentation is strong; GFSI from scratch can take 12-18 months).

## 2. Gate Definitions and Deliverables

### G0 — Scoping & Feasibility

**Entry Criteria:**
- Business need identified (customer requirement, market access, strategic goal)
- Executive sponsor assigned

**Key Activities:**
- Define which certification(s) to pursue
- Research certifier options (load `certifier_directory.md`)
- Develop rough budget estimate
- Develop rough timeline
- Identify project team and assign project manager
- Assess organizational readiness (do we have QA staff? Document control? HACCP expertise?)

**Deliverables:**
- Certification project charter (1-2 pages): scope, objectives, budget, timeline, team
- Certifier shortlist (2-3 options with pros/cons)
- Preliminary cost estimate table

**Go/No-Go Decision:**
- Does the business case justify the investment?
- Do we have the resources (people, budget, time)?
- Is the timeline feasible given customer deadlines?

**Decision Owner:** Operations Director or VP Operations

---

### G1 — Gap Assessment Complete

**Entry Criteria:**
- G0 approved
- Facility access granted for assessment
- Current documentation (formulas, SOPs, supplier list, floor plans) collected

**Key Activities:**
- Conduct gap assessment against target certification standard
  - For NOP: compare current practices to 7 CFR Part 205 handler/producer requirements
  - For GFSI: compare to specific scheme requirements (SQF Code, BRC Issue 9, etc.)
  - For specialty: compare to specific program requirements (Non-GMO Technical Standard, GFCO requirements, etc.)
- Review all ingredients for compliance (National List for organic, high-risk crop list for Non-GMO, ingredient review for Kosher/Halal, gluten risk for GFCO)
- Map current supplier documentation status
- Identify training gaps
- Assess facility/equipment suitability
- Build risk register

**Deliverables:**
- Written gap analysis report (requirement | current state | gap | priority | estimated effort)
- Ingredient compliance matrix
- Supplier documentation status log
- Risk register (see template below)
- Updated timeline and budget based on gaps identified

**Go/No-Go Decision:**
- Are the gaps closable within the target timeline?
- Are there any showstoppers (prohibited equipment, fundamentally non-compliant ingredient, facility design flaw)?
- Is the revised budget still justified?

**Decision Owner:** QA Director

---

### G2 — Program Build Complete

**Entry Criteria:**
- G1 approved
- Gap report reviewed and accepted
- Budget for program build approved

**Key Activities:**
- Write or update Organic System Plan (for NOP) or Quality Management System documentation (for GFSI)
- Develop/revise SOPs:
  - Receiving and ingredient inspection
  - Organic/specialty ingredient handling and segregation
  - Production procedures
  - Cleaning and sanitation
  - Pest management
  - Labeling and packaging
  - Mass balance / audit trail
  - Complaint handling
  - Recall/withdrawal
  - Supplier approval and monitoring
- Collect all supplier certifications (organic certificates, Non-GMO affidavits, Kosher letters, allergen statements)
- Conduct employee training
- Implement any facility modifications identified in gap assessment
- Set up document control system
- Set up recordkeeping templates and systems

**Deliverables:**
- Complete OSP or QMS documentation package
- All SOPs written and approved
- Supplier certificate binder (or digital equivalent) complete
- Training records for all relevant employees
- Facility modifications completed and documented
- Document control system operational
- Recordkeeping templates deployed

**Go/No-Go Decision:**
- Are all program elements documented?
- Are all supplier certs collected and current?
- Is training complete?
- Are facility modifications done?

**Decision Owner:** QA Director + Operations Manager

---

### G3 — Internal Audit Passed

**Entry Criteria:**
- G2 approved
- Minimum 2-4 weeks of operating records generated under new system

**Key Activities:**
- Conduct internal audit against certification standard
  - Audit checklist follows the certification standard structure
  - Review documentation, interview staff, observe operations, verify records
- Document all findings (observations, minors, majors)
- Develop corrective action plan for all findings
- Implement corrective actions
- Verify closure of all corrective actions
- Conduct mock recall/traceability exercise
- Stage evidence binder for external audit

**Deliverables:**
- Internal audit report
- Corrective action log with closure evidence
- Mock recall report (if applicable)
- Evidence binder ready for external auditor/inspector

**Go/No-Go Decision:**
- Are all audit findings closed?
- Does the mock recall meet requirements (SQF: 100% trace within 4 hours; BRC: within 4 hours)?
- Is the evidence binder complete?
- Are staff confident and prepared?

**Decision Owner:** QA Director + Certifier Liaison

---

### G4 — Application Submitted

**Entry Criteria:**
- G3 approved
- Internal audit passed with all findings closed
- Certifier selected (from G0 shortlist, refined through process)

**Key Activities:**
- Finalize certifier selection
- Complete application forms
- Submit application package:
  - For NOP: OSP, facility description, product list, ingredient list, supplier certificates, fee payment
  - For GFSI: application form, scope definition, fee payment, QMS documentation (if required by CB)
  - For specialty: varies by program
- Pay application fees
- Receive application acknowledgment and inspector/auditor assignment

**Deliverables:**
- Completed application package
- Payment confirmation
- Inspector/auditor assignment (date and name, when received)
- Pre-audit communication log

**Go/No-Go Decision:**
- Is the application complete and accepted by the certifier?
- Is the audit/inspection date confirmed?
- Are there any scheduling conflicts or resource gaps for the audit date?

**Decision Owner:** QA Director

---

### G5 — Inspection/Audit Passed

**Entry Criteria:**
- G4 approved
- Audit/inspection date confirmed
- All personnel available and briefed

**Key Activities:**
- Prepare facility for inspection/audit (not "cleaning up" — but ensuring day-of-audit logistics are handled)
- Opening meeting with inspector/auditor
- Facility tour and production observation
- Document review (records, logs, OSP/QMS)
- Staff interviews
- Closing meeting — receive preliminary findings
- Receive formal inspection/audit report
- Develop corrective action response for any findings
- Submit corrective actions to certifier within required timeframe

**Deliverables:**
- Inspection/audit report (from certifier)
- Corrective action responses (with evidence) submitted
- Certifier acknowledgment of corrective action acceptance

**Go/No-Go Decision:**
- Did we pass the audit?
- Are all corrective actions accepted by the certifier?
- Is the certifier ready to issue the certificate?

**Decision Owner:** QA Director + Executive Sponsor

---

### G6 — Certificate Issued

**Entry Criteria:**
- G5 approved (audit passed, corrective actions accepted)
- Certifier grants certification

**Key Activities:**
- Receive certificate
- Verify certificate details (scope, products, dates, operation name)
- Conduct label review (especially for organic — verify label category, seal usage, certifier logo placement)
- Notify customers and buyers
- Update marketing materials
- Set up surveillance/renewal calendar
- Establish ongoing compliance monitoring system
- Celebrate the team!

**Deliverables:**
- Certificate (hard copy and digital)
- Label review completed and approved
- Customer notification letters/emails sent
- Surveillance/renewal calendar set (with 90-day advance reminders)
- Ongoing compliance owner assigned

**Decision Owner:** QA Director + Marketing

## 6. Common Schedule Killers

### Ingredient Supplier Delays (most common)
- Suppliers slow to provide organic certificates, Non-GMO affidavits, allergen statements
- Supplier certificates expired and supplier doesn't respond to renewal requests
- **Mitigation:** Start supplier outreach at G1, not G2. Set hard deadlines. Identify backup suppliers.

### National List Substance Review
- Determining whether a specific substance is National List-compliant can require weeks of research
- OMRI listing helps but doesn't resolve all ambiguity
- Certifier may need to make formal determination
- **Mitigation:** Submit substance questions to certifier early in G1/G2. Use OMRI database as first screen.

### Certifier Application Backlogs
- Peak season (spring for producers, Q3-Q4 for handlers preparing for new year) creates backlogs
- Inspector availability in certain geographies is limited
- **Mitigation:** Apply early. Discuss timeline expectations with certifier during G0 shortlisting.

### Facility Modifications
- Construction/renovation takes longer than expected
- Permits required for certain modifications
- Equipment lead times
- **Mitigation:** Identify facility needs at G1. Start contractor/equipment procurement at G2 start, not end.

### Staff Turnover
- Key QA person leaves mid-project
- SQF Practitioner departs (must have a trained replacement)
- **Mitigation:** Cross-train at minimum 2 people on every critical function.

### Scope Creep
- "While we're at it, let's add three more SKUs" — each additional product adds ingredient review, supplier documentation, and possibly testing
- **Mitigation:** Lock product scope at G0. Any additions go through formal scope change with timeline impact assessment.

### Corrective Action Rejections
- Certifier rejects corrective actions as insufficient
- Multiple rounds of back-and-forth
- **Mitigation:** Write thorough CAs with root cause, corrective action, verification, and prevention. Pre-review with consultant.

## 7. Go/No-Go Decision Criteria

### Gate Review Meeting Format
1. Project manager presents gate deliverables status
2. Risk register review
3. Budget vs. actual review
4. Timeline vs. plan review
5. Open discussion of issues
6. Go/No-Go/Conditional Go decision

### Decision Options
- **Go:** All deliverables met. Advance to next gate.
- **Conditional Go:** Minor items outstanding but not blocking. Advance with conditions (e.g., "Go to G3 but complete supplier cert X within 2 weeks").
- **No-Go:** Critical deliverables incomplete. Remain at current gate. Assign actions and reconvene.
- **Kill:** Business case no longer valid. Cancel project. (Rare but important option to have.)

## 8. Multi-Certification Sequencing

### Common Stacks and Recommended Sequencing

**Organic + Non-GMO + Kosher (common for natural/organic food brands)**
1. Start Kosher first (fastest, simplest, and ingredient review informs organic/Non-GMO)
2. Organic and Non-GMO in parallel (significant documentation overlap — ingredient sourcing, supplier documentation)
3. Non-GMO often slightly faster than organic, so may complete first

**Organic + GFSI (SQF or BRC) (common for brands selling to major retailers)**
1. Start GFSI first (takes longest, broadest scope)
2. Organic in parallel once GFSI G2 is underway (organic is narrower in scope but has unique requirements)
3. GFSI documentation system provides foundation for organic recordkeeping

**Organic + Fair Trade (common for coffee, cocoa, tea brands)**
1. Start organic first (federal law, more rigid requirements)
2. Fair Trade after organic is in G2+ (supply chain already being documented for organic)
3. Fair Trade sourcing commitments can be built into organic supplier program

**GFSI + Gluten-Free (common for manufacturers with allergen-sensitive products)**
1. Start GFSI first (allergen management is a major GFSI requirement — building this well supports GFCO)
2. GFCO/NSF after GFSI G2 (leverage allergen management program already built)

### Parallel vs. Sequential Decision Framework
- **Parallel if:** dedicated project manager, sufficient QA staff (2+ people), budget allows, timeline is tight
- **Sequential if:** small team (1 QA person), limited budget, no timeline pressure, organization new to certification management
- **Key principle:** The first certification is the hardest. Each subsequent certification has significant documentation and process overlap. Plan for this synergy.
